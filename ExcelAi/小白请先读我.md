# 功能效果

1. 使用Excel的佬，请看Excel子目录进行安装，两个文件需要放在一起，点击msi或exe进行安装
2. 使用WPS的佬，请看WPS子目录进行安装，WPS限制较Excel多，所以进入后还需要看一下操作方式
3. windows系统要求  支持.NET Framework 4.7.2
4. 如果需要卸载，可以在windows控制面板中找到应用，应用和功能，按驱动器搜索，然后卸载
